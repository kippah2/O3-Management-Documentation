ChangeManagerment.ps1

The change management script is responsible for comparing the most recent active directory-pulled user list with the user list that was last used for the user prov script.

After User-Prov runs, place its input get-users.csv into the “old input” folder of the change management script. The get-ad-users script should be run before this script, ideally on the day of. Place the output of the most recent get-users.csv into the “new input” folder of the change management script.

The change management script also takes another input from the running of the user prov script. This file is called CollaborationInfo.csv. It looks like:

It then creates an object array based on the discrepancies which contains objects for every instance where the action was “Changed Manager”. Then, the script makes another object array called $collabinfo, which contains every entry from CollaborationInfo.csv where “folderOwner” does not equal “addedEmail”.

1$discrep = $discrepancies | Where-Object {$_.Action -contains "Manager Changed"}
2$collabinfo = $collaborations | Where-Object {$_.folderOwner -ne $_.email}

Once the full array is made, the script iterates through each object, comparing to every object in $collabinfo (nested foreach-object loop). When comparing to the CollaborationInfo.csv file, it will be looking for matches between the $collabinfo.folderOwner value and $discrepancies.email.

1foreach ($discrepobject in $discrep) {
2foreach ($collabobject in $collabinfo) {

If a match is found, it will then looks at $collabinfo.managerEmail and compares against $discrep.oldManagerEmail and $discrep.newManagerEmail.

If a new collaboration is being added, all of the relevant information will be appended to the CollaborationInfo.csv file. If a collaboration is being deleted, the corresponding line in the CollaborationInfol.csv file will be deleted.

Here is the script in it’s entirety:

changemanagement.ps1

  1# Define the paths to the input CSV files
  2$latestFile = ".\newInput\get-users-test.csv"
  3$secondLatestFile = ".\oldInput\get-users.csv"
  4
  5# Define the path to the collaboration CSV file
  6$collaborationFile = ".\newInput\CollaborationInfo.csv"
  7
  8# Load the newest and second newest CSV files into variables and convert them to hash tables
  9$oldUsers = @{}
 10$newUsers = @{}
 11
 12# Function to create a custom object for a user
 13function CreateUserObject($user) {
 14    $name = $user.firstName + " " + $user.lastName
 15    [PSCustomObject]@{
 16        name = $name
 17        username = $user.username
 18        email = $user.email
 19        manager = $user.manager
 20        managerEmail = $user.managerEmail
 21    }
 22}
 23
 24Import-Csv $secondLatestFile | ForEach-Object { $oldUsers[$_.username] = CreateUserObject $_ }
 25Import-Csv $latestFile | ForEach-Object { $newUsers[$_.username] = CreateUserObject $_ }
 26
 27# Load the collaboration data from the CSV file
 28$collaborations = Import-Csv $collaborationFile
 29
 30# Create an array to store discrepancies
 31$discrepancies = @()
 32
 33# Compare old and new users
 34foreach ($oldUser in $oldUsers.GetEnumerator()) {
 35    $username = $oldUser.Key
 36    $oldUserInfo = $oldUser.Value
 37    $newUserInfo = $newUsers[$username]
 38
 39    if (!$newUserInfo) {
 40        # User from the old file is not present in the new file
 41        $discrepancies += [PSCustomObject]@{
 42            Action = "User Removed"
 43            name = $oldUserInfo.name
 44            username = $oldUserInfo.username
 45            email = $oldUserInfo.email
 46            oldManager = $oldUserInfo.manager
 47            oldManagerEmail = $oldUserInfo.managerEmail
 48            newManager = $null  # Set to $null when user is removed
 49            newManagerEmail = $null  # Set to $null when user is removed
 50        }
 51    } else {
 52        # User exists in both files, check for manager and managerEmail changes
 53        if ($oldUserInfo.manager -ne $newUserInfo.manager -or $oldUserInfo.managerEmail -ne $newUserInfo.managerEmail) {
 54            $discrepancies += [PSCustomObject]@{
 55                Action = "Manager Changed"
 56                name = $oldUserInfo.name
 57                username = $oldUserInfo.username
 58                email = $oldUserInfo.email
 59                oldManager = $oldUserInfo.manager
 60                oldManagerEmail = $oldUserInfo.managerEmail
 61                newManager = $newUserInfo.manager
 62                newManagerEmail = $newUserInfo.managerEmail
 63            }
 64
 65
 66        } else {
 67            # If the user's manager or managerEmail did not change, add the old values as well
 68            $discrepancies += [PSCustomObject]@{
 69                Action = "No Change"
 70                name = $oldUserInfo.name
 71                username = $oldUserInfo.username
 72                email = $oldUserInfo.email
 73                oldManager = $oldUserInfo.manager
 74                oldManagerEmail = $oldUserInfo.managerEmail
 75                newManager = $null  # Set to $null when there is no change
 76                newManagerEmail = $null  # Set to $null when there is no change
 77            }
 78        }
 79    }
 80}
 81
 82# Display the discrepancies
 83$discrepancies | Format-Table -Property Action, name, email, oldManager, oldManagerEmail, newManager, newManagerEmail -AutoSize
 84
 85# Export the discrepancies to a CSV file
 86$discrepancies | Export-Csv -Path ".\Output\discrepancies.csv" -NoTypeInformation
 87
 88$discrep = $discrepancies | Where-Object {$_.Action -contains "Manager Changed"}
 89$collabinfo = $collaborations | Where-Object {$_.folderOwner -ne $_.email}
 90
 91makeChanges $discrep $collabinfo
 92
 93# Function for adding new managers to folders and removing old managers from folders.
 94function makeChanges ($discrep, $collabinfo){
 95    # Iterate through $discrep, which is object array of instances where the manager changed.
 96    foreach ($discrepobject in $discrep) {
 97        # Iterate through $collabobject, which is an object array of collaborations where the owner of the folder is not the same person who the collaboration is for.
 98        foreach ($collabobject in $collabinfo) {
 99            echo "collab: $($collabobject.email) discrep: $($discrepobject.email)"
100            if ($collabobject.folderOwner -eq $discrepobject.email) {
101                Write-Host "Match found for $($collabobject.folderOwner) and $($discrepobject.email)"
102                if ($collabobject.email -ne $discrepobject.newManagerEmail) {
103                    $newManagerResp = (box folders:collaborations:add $collabobject.folderID --role editor --login $($discrepobject.newManagerEmail) --json 2>&1)
104                    $newManagerRespObj = $newManagerResp | ConvertFrom-JSON
105                    $CollaborationID = $newManagerRespObj.id
106                    $folderName = $newManagerRespObj.item.name
107                    $newcollaborations += [PSCustomObject]@{
108                        folderowner = $collabobject.folderOwner
109                        folderName = $collabobject.folderName
110                        added = $discrepobject.newManager
111                        email = $discrepobject.newManagerEmail
112                        collaborationID = $CollaborationID
113                        folderID = $collabobject.folderID
114                    } | Export-Csv -Path $collaborationFile -NoTypeInformation -Append
115                    Write-Host "Added collaborator $($discrepobject.newManager) to $folderName."
116                }
117                if ($collabobject.email -eq $discrepobject.oldManagerEmail) {
118                    $deleteManagerResp = (box collaborations:delete $collabobject.collaborationID)
119                    Write-Host "Removing collaboration with ID $collaborationID"
120                    # Remove the collaboration from the $collaborations array
121                    $global:collaborations = $global:collaborations | Where-Object { $_.collaborationID -ne $collabobject.collaborationID }
122                    # Export the updated $collaborations array back to the CSV file
123                    $global:collaborations | Export-Csv $collaborationFile -NoTypeInformation
124                }
125            else {
126                Write-Host "Folder owner $($collabobject.folderOwner) is not user from discrepancy $($discrepobject.email). Moving on."
127            }
128        }
129    }
130    }
131}
